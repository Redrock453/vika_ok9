## 🚀 ПЛАН ДЕЙСТВИЙ "ДЕНЬ 1" (СЕГОДНЯ)

### ЭТАП 1: Подготовка фундамента (WSL 2)
1. Запустить PowerShell от Админа.
2. Выполнить: `wsl --install`.
3. **ПЕРЕЗАГРУЗИТЬ ПК.** (Критический шаг).

### ЭТАП 2: Установка Docker Desktop
1. Скачать: https://www.docker.com/products/docker-desktop/
2. При установке убедиться, что выбрано **WSL 2 backend**.
3. После установки запустить Docker и дождаться статуса "Running".

### ЭТАП 3: Запуск Qdrant (Память системы)
1. Выполнить в терминале:
   ```bash
   docker pull qdrant/qdrant
   docker run -p 6333:6333 -p 6334:6334 -v qdrant_storage:/qdrant/storage qdrant/qdrant
   ```
2. Проверить: `curl http://localhost:6333/health`.
